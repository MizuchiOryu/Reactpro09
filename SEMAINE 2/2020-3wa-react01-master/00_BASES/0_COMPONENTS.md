# JSX

## Notions nécessaires
- ES6 classes


## Nouvelles notions
- React Component
- Functional Component / Stateless Component


3 propriétés à toujours avoir en tête :

- Configurables (props)
- Réutilisables (Effets de bord contrôlés)
- Imbriquables (Indépendants de la hiérarchie)
